TurboWarp Privacy Policy

Updated October 11th, 2020.

Data collection

We may collect web server logs which can include IP addresses, user agents, pages visited, timestamps, and other statistics. These logs are deleted within 14 days.

A self-hosted instance of Plausible Analytics is used to track website usage. You can see the information collected by this on https://stats.turbowarp.org/turbowarp.org. This server stores no personal information.

When loading projects that contain "cloud variables", your IP address, supplied username, and project ID are logged. This information is deleted within 14 days. The data stored in cloud variables is purged within 24 hours of no users being connected to a given project.

Sharing information

Personal information is never shared, unless necessary to provide the service (eg. complying with local laws).

Third Parties

This privacy policy does not apply to third parties. A non exhaustive list of notable third parties include:

Scratch

Projects may be loaded from the Scratch website. Read the Scratch privacy policy.

jsDelivr CDN

The Scratch 3 Dev Tools are loaded from the jsDelivr CDN. Read the jsDelivr CDN privacy policy.

Contact

Any concerns related to privacy or any other matter should be sent to: contact@turbowarp.org
